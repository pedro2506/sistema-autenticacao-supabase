Teste de Navegação

Clique nos links abaixo para testar a navegação:

Ir para Login Ir para Cadastro

Instruções para teste:

  1. Clique em "Ir para Login" - deve abrir a página de login
  2. Na página de login, clique em "Cadastre-se" - deve ir para a página de cadastro
  3. Na página de cadastro, clique em "Faça login" - deve voltar para a página de login
  4. Teste o formulário de cadastro preenchendo os campos

Para debug:

Abra o console do navegador (F12) para ver as mensagens de debug.
